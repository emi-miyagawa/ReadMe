
# Operations Automation (GAS)

## Overview
This repository contains small automation scripts built to reduce manual operational workload.

The focus is not full automation, but:
- reducing repetitive tasks
- improving accuracy
- supporting existing processes under real constraints

---

## Examples

### 1. Address Validation
- Validates input data before processing
- Reduces downstream errors

### 2. Email Automation
- Automates repetitive notification workflows
- Reduces manual communication effort

### 3. Gift Card Processing
- Supports structured handling of transaction data
- Improves processing consistency

---

## Design Approach

In many cases, full automation is not suitable due to variability and exceptions.

Instead, I focus on:
- partial automation
- human-in-the-loop processes
- stability over complexity

---

## Positioning

This repository demonstrates how I use lightweight automation to support operations design.

It is not about building complex systems, but about making operations more reliable and scalable.
